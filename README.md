

📝 Formulário Web Responsivo

Este projeto consiste em um formulário web simples e elegante, desenvolvido utilizando HTML e CSS. O objetivo é praticar a criação de interfaces de entrada de dados com foco em usabilidade, organização e design responsivo.

📌 Funcionalidades
📋 Campos de formulário (nome, email, senha, etc.)
✅ Validação básica com HTML5
🎨 Estilização moderna com CSS
📱 Layout responsivo para diferentes dispositivos
🔘 Botões interativos (hover e foco)
🛠️ Tecnologias utilizadas
HTML5
CSS3
📂 Estrutura do projeto
/formulario-web
│── index.html
│── /css
│     └── styles.css
🚀 Como executar o projeto
Clone este repositório:
git clone https://github.com/math0410/Projeto-Formulario

Acesse a pasta do projeto:
cd formulario-web
Abra o arquivo index.html no navegador:
Clique duas vezes no arquivo
ou
Utilize uma extensão como Live Server no VS Code
🎯 Objetivo do projeto

Este projeto foi criado com fins educacionais para praticar:

Estruturação de formulários com HTML
Uso de inputs e validações nativas
Estilização com CSS
Criação de layouts responsivos
💡 Melhorias futuras
Validação com JavaScript
Integração com backend (envio de dados)
Feedback visual de erros em tempo real
Máscaras de input (telefone, CPF, etc.)
Tema escuro (dark mode)
